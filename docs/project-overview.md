# AGCH-Impl 專案概覽

## 專案簡介

**AGCH-Impl** 是 AGCH（Aggregation-based Graph Convolutional Hashing）論文的實作專案，旨在實現無監督跨模態檢索系統。

---

## 技術背景

### 問題定義

在多媒體數據時代，跨模態檢索（如以圖搜文、以文搜圖）面臨以下挑戰：
- **語義導引不足**: 無監督場景缺乏人工標註
- **模態異質性**: 不同模態（影像/文本）的特徵空間差異大
- **單一指標偏誤**: 傳統方法僅依賴餘弦相似度

### AGCH 解決方案

AGCH 透過三大技術支柱解決上述問題：

1. **聚合相似度矩陣**: Hadamard 乘積融合方向相關性與維度差異
2. **多模態深度編碼**: CNN + MLP 提取非線性特徵
3. **GCN 結構學習**: 利用鄰域聚合挖掘數據流形結構

## 系統架構

### 網路組件

| 組件 | 結構 | 輸出維度 | 激活函數 |
|------|------|----------|----------|
| 影像編碼器 | AlexNet (fc-7) | c 維 | ReLU → tanh |
| 文本編碼器 | 3層 MLP | K → 4096 → c | ReLU → tanh |
| GCN 模組 | 2層 GCN + FC | 4096 → 2048 → c | ReLU → tanh |
| 融合模組 | 全連接層 | c 維 | tanh |

### 核心公式

**聚合相似度矩陣構建:**

```
S_ij = C_ij · D_ij
```

其中:
- `C_ij`: 餘弦相似度（方向相關性）
- `D_ij`: 歐幾里得距離指數變換（維度差異）
